HTML-CSS

HTML e CSS

Todas as tags HTML que vimos durante as aulas.
Tag Usei ? Descrição
body Sim Essa tag define o corpo da página. A página toda esta contida nela.
h1 ... h6 Sim Essa tag é usada para definir cabeçalhos. O h1 foi usado no menu e o h3 é usado para dar o titulo de cada página.
p Sim Essa tag define um parágrafo. Uso em todas as páginas.
br Sim Cria uma quebra de linha. usei para organizar algumas páginas.
hr Sim Cria uma linha horizontal. usei na página inicial.
center Não Centra o texto
blockquote Sim Cria um bloco de texto citado. usei para mostrar o poema.
pre Não Apresenta o texto tal como digitado no documento.
code Não Usa uma fonte de código, mas não adiciona nem uma cor diferente de acordo com a sintaxe.
ol Não Cria uma lista ordenada.
ul Sim Cria uma lista não ordenada. Usei ele no index para explicar o website.
li Sim Cria um item da lista (Deve ser usado dentro de um ul ou ol). Usei ele dentro do ul no index.
dl Não Cria uma lista de descrições.
dt Não O termo que vai ser descrito.
dd Não Descrição do termo.
img Sim Uma tag que tem uma imagem associada. Usei no menu e em algumas páginas.
figure Sim Permite agrupar um img com uma figcaption. Usei na página de multimedia
a Sim Cria um hiperlink. Usei no menu e em algumas outras páginas.
iframe Sim Permite incorporar outro documento no documento. Usei na página de multimedia
table Sim Cria uma tabela. Usei na página de info.
DOCTYPE Sim Indentifica a versão do HTML
head Sim Usado para agrupar outras tags de configuração da página HTML
title Sim Define o titulo da página
meta Sim Define características do documento.
style Sim Usado para embeber uma StyleSheet no documento.
link Sim Estabelece relações entre o documento corrente e outros documentos.
base Não Define o endereço base para recursos indicados no documento.
script Não Embebe scripts no documento
form Sim Usado para agrupar outras tags que formam o formulário. Usei na página de quiz e comentário
input Sim Usado para recolher input do utilizador (Existem diversos tipos de input)
textarea Sim Define um campo de entrada multi-linhas.
fieldset Sim Agrupa elementos do formulário em uma caixa. usei no quiz
output Não Apresenta o valor de um oninput
button Não Define um botão clicável
svg Não Cria um contentor de gráficos SVG
Todos o seletores CSS que vimos durante as aulas.
Seletor Usei ? Descrição
Universal Sim Aplica o estilo a todos os elementos da página.
Tipo Sim A regra será aplicada a todos os elementos que possuam esse marcador
Classe Sim A regra será aplicada a todos os elementos que tiverem essa classe
Id Sim A regra será aplicada para o elemento que tiver esse id especifico.
Atributo Não A regra se aplica a atributos de elementos
Pseudo Classes Sim Seletores aplicados a estados de elemntos
Pseudo-Elementos Não Regras que se aplicam a partes do elemento
Todos os combinadores de seletores CSS que vimos durante as aulas.
Combinador Usei ? Descrição
Descendentes Sim Todos os elementos que estão dentro do primeiro elemento
Filhos Sim Todos os elementos que são filhos do primeiro elemento
Adjacentes Não Todos os elementos que estão colocados depois do primeiro elemento
Imediatamente Adjacentes Não Todos os elementos que estão colocados "imediatamente" depois do primeiro elemento
Agrupar Sim Diferentes seletores separados por virgula
